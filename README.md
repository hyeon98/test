# 가계부 서비스

## **프로젝트 개요**

- **프로젝트 개요**  
  사용자의 수입/지출을 기록하고, 다양한 조건으로 조회 및 분석할 수 있는 개인 가계부 웹 애플리케이션입니다.

- **프로젝트 목표**  
  가계부 서비스 앱 개발을 통해 수업에서 학습한 기술 요소를 실제 서비스에 적용하고,  
  실무 중심의 개발 경험과 협업 역량을 동시에 향상시키는 것을 목표로 합니다.
  1. **프로그래밍 능력 향상**  
     수업에서 배운 기술을 활용하여 서비스를 개발하고, 작성한 코드에 대해 스스로 이해하고 설명할 수 있는 수준을 목표로 합니다.  
     또한 데이터 흐름과 비즈니스 로직을 직접 구현하며 개념적 이해를 실무 역량으로 확장합니다.

  1. **전체 개발 프로세스 경험 향상**  
     기획 → 요구사항 정의 → DB 설계 → 구현 → 배포까지, 소프트웨어 개발의 전체 생애주기를 경험합니다.  
     이를 통해 각 단계 간의 유기적인 연결을 이해합니다.

  1. **협업 및 커뮤니케이션 역량 향상**  
     팀 프로젝트를 통해 역할 분담, 의사소통, 의견 조율 과정을 경험하며 협업 능력을 향상시킵니다.  
     단순한 개인 개발을 넘어, 함께 일할 수 있는 개발자로 성장하는 것을 목표로 합니다.

  1. **체계적인 개발 관리 경험**  
     Git을 활용한 형상 관리 및 협업 방식을 적용하고, 팀 단위 개발 컨벤션을 수립합니다.  
     이를 통해 체계적인 개발 환경과 협업 기반 개발 역량을 강화합니다.

  1. **실전 프로젝트 대비**  
     이후 진행될 파이널 프로젝트를 대비하여, 팀 프로젝트 과정에서의 시행착오를 미리 경험하고 개발 자신감을 확보합니다.

## 대표 기능

1. 수입 / 지출 내역 등록 및 관리
2. 거래 내역 조회 (필터링 / 정렬)
3. 월별 재정 요약 (수입, 지출, 순이익)
4. 카테고리별 소비 분석
5. 예산 설정 및 초과 경고 알림

## 프로젝트 요구사항

[요구사항 명세](docs/requirements.md)

### 기능 요구사항

- **거래 기록 관리**
  - 수입 / 지출 내역 등록
  - 날짜, 금액, 카테고리, 메모, 거래 유형 입력
  - 입력값 검증 및 저장 처리
- **거래 내역 조회**
  - 전체 거래 리스트 조회
  - 기간, 카테고리, 거래 유형별 필터링
  - 날짜/금액 기준 정렬 기능
- **월별 재정 요약**
  - 월별 총 수입 / 총 지출 / 순이익 제공
  - 카테고리별 지출 통계 제공
- **예산 관리**
  - 월별 예산 설정 및 수정
  - 예산 대비 지출 현황 및 소진율 확인
- **예산 경고 알림**
  - 예산 사용량 기준 경고 메시지 제공
  - 예산 초과 시 알림 표시

### 비기능 요구사항

- **성능**
  - 필터 및 정렬 시 빠른 응답 속도 유지
  - 대량 데이터에서도 안정적인 렌더링
- **보안**
  - 입력값 검증을 통한 데이터 무결성 유지
  - 클라이언트 측 기본적인 유효성 검사 적용

## **설치 및 실행 방법**

이 프로젝트를 로컬에서 실행하기 위해선 다음의 설치 및 실행 과정을 따라주세요:

1. **프로젝트 클론하기**

   ```bash
   git clone https://github.com/SongHyeongJin/expense-tracker.git
   ```

2. **디렉토리로 이동하기**

   ```bash
   cd expense-tracker
   ```

3. **패키지 설치**

   ```bash
   npm install
   ```

4. **JSON 서버 및 프로젝트 실행하기**

   ```bash
   # JSON 서버 실행 (mock API)
   npx json-server --watch db.json --port 3000
   ```

   새로운 bash 열기.

   ```bash
   # 개발 서버 실행
   npm run dev
   ```

5. **구동 후 확인하기**
   - 웹 브라우저를 열고 [http://localhost:5173](http://localhost:5173/) 에 접속해 확인합니다.
